# Fleet & Driver Management - Admin Panel

This repository contains the web-based admin portal for the Fleet and Driver Management system.

## Features
- **Assign Trips**: Create and assign routes to drivers and vehicles.
- **Document Review**: Approve or reject driver KYC documents.
- **Fleet Reports**: View analytics for trips, drivers, and billing.
- **Live Tracking**: Monitor active trips on a live map.
- **Route Master**: Manage pre-defined routes and locations.
- **Advance Approvals**: Manage and approve driver advance requests.

## Development
Runs on Flutter Web.
```bash
flutter run -d chrome
```
